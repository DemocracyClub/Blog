Use our data

We want to make all our data as easy to use as possible.

Please talk to us if you have any particular requirements, or are after some advice on UK election data of any sort.

Open Electoral Commission

This is a very unfinished work in progress, but it already offers a list of UK political parties in JSON format.

This includes party emblems, names of various party officer and addresses.

All the data is scraped from the ElectoralCommission web site because their site doesn't allow deep linking or any form of machine readable version

Browse the list of parties or consume the JSON

YourNextMP.com

Backed by the Popolo data standard using mySociety's PopIt backend, YourNextMP.com is the high quality source of free UK candidate data.

We currently have data for the 2010 and 2015 General Elections and hope to add more elections in the future.

The data is under CC BY-SA 4.0. Find out more at the Open Data API page

Election Leaflets

We have a huge archive of election leaflets that is ready for anyone to analyse. Please contact us if you would like to know more
